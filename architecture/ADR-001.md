# ADR-001

# Core Platform Strategy

## Status

Accepted

## Context

Building every engineering tool from scratch will
increase cost, time and maintenance.

## Decision

TECTRONIX FLOW will focus on AI, Knowledge and
Engineering Workflow.

Existing mature tools will be integrated instead of rebuilt.

## Core Integrations

- tscircuit
- XOD
- KiCad
- ngspice
- PlatformIO

## Principles

- Build Intelligence
- Integrate Best Tools
- Open Architecture
- AI Native
- Knowledge First
